FCC - URL Shortener Microservice

Using clementine.js for FCC

User stories

  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Create shortcut

http://thcn-url.herokuapp.com/new/http://vnexpress.net

Result return

{'origin':'http://vnexpress.net', 'short': 1234}

Using shortcut

http://thcn-url.herokuapp.com/1234
